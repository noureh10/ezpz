Usage:
ezpz [FLAGS] [COMMANDS] [~PATH]

To configure ezpz, please do as follow:

   ezpz setconfig

Once you achieved that, you can launch the different set of commands:
   setconfig    set a configuration file that ezpz will be working on.
   start        start the project zomboid server
   interactive  launch interactive manager mode.
   status       get the ongoing status of the server.
   modlist      get the list of mods installed to the server.
   version      show the current running version of ezpz.
   update       update ezpz to the latest version.
   help         show the general help panel.
   refresh      this command has for sole purpose to get back to its original status.
                By doing this, you ensure that the application clears every data.
Available flags:
   -v           verbose mode, useful to track underlying issues with some actions performed

   The CLI is silent by default.
